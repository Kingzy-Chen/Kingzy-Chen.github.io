【LeetCode 笔记-Array 数组-02】No.27 移除元素

发表于 2021-03-16 更新于 2021-03-23 分类于 LeetCode 笔记 ， Array 数组 Valine：
本文字数： 599 阅读时长 ≈ 1 分钟

题目描述

给定一个数组 nums 和一个值 val，你需要 原地 移除所有数值等于 val 的元素，并返回移除后数组的新长度。不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并 原地修改输入数组 。元素的顺序可以改变。你不需要考虑数组中超出新长度后面的元素。

示例 1 示例 2
输入: nums = [3, 2, 2, 3], val = 3
输出: 2, nums = [2, 2]		 输入: nums = [0, 1, 2, 2, 3, 0, 4, 2], val = 2
输出: 5, nums = [0, 1, 4, 0, 3]

解题思路

本题使用快慢指针法较为简便。

快慢指针法

快慢指针法顾名思义，即设置 fast 和 slow 快慢两个指针。两者的移动速度被设置为不同，从而达到一些巧妙的效果。当然，本题中快慢指针法的应用较为简单，但不失为一种重要的思想。
