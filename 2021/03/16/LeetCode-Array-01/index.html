【LeetCode 笔记-Array 数组-01】No.35 搜索插入位置

发表于 2021-03-16 更新于 2021-03-23 分类于 LeetCode 笔记 ， Array 数组 Valine：
本文字数： 1.1k 阅读时长 ≈ 1 分钟

题目描述

给定一个 排序数组 和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。你可以假设数组中 无重复元素。

示例 1 示例 2 示例 3 示例 4
输入: [1, 3, 5, 6], 5
输出: 2		 输入: [1, 3, 5, 6], 2
输出: 1		 输入: [1, 3, 5, 6], 7
输出: 4		 输入: [1, 3, 5, 6], 0
输出: 0

解题思路

本题有两种解题思路：暴力法和二分查找法。

暴力法

因为题目给定的是排序数组，所以可以直接遍历数组并根据大小关系得到目标位置。

时间复杂度：O(n)
空间复杂度：O(1)

二分查找法

二分查找法的思想在于设置 left 和 right 头尾两个指针，并结合中间指针 mid 将数组划分为左右两个区间。每次循环时，根据 mid 指向的元素大小判断目标位置在左区间还是右区间。由此反复更新区间，最终确定目标的位置所在。

时间复杂度：O(log n)
空间复杂度：O(1)
